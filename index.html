Página Web con Interacciones utilizando Doom

Instrucciones de Teclas

  • → : Abrir una imagen
  • Esc: Cerrar la imagen
  • ↑ : Crear una tabla 3x3 con "X" en el centro de color azul
  • A: Mostrar el título de la página web
  • Barra espaciadora: Enviar un correo electrónico
  • Q: Ejecutar un script
  • O: Realizar una consulta a la base de datos (no es posible desde el navegador)
